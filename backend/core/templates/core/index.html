API

Endpoints disponíveis sob /api/:

  • POST /api/register — cadastrar usuário
  • POST /api/login — autenticar e obter token
  • GET/POST /api/grupos
  • GET/POST /api/atletas
  • GET/POST /api/campos
  • GET/POST /api/horarios
  • GET/POST /api/partidas
